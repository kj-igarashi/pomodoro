# Pomodoro（Mac メニューバー常駐アプリ）

作業/休憩を自由に設定し、work→rest→work→rest… と無限に繰り返すシンプルなポモドーロタイマー。
Mac備え付けの音で区切りを知らせます。

## 特長
- メニューバーに残り時間を常時表示（🍅=作業 / ☕️=休憩）
- 作業/休憩の分数はプリセット（25/5・50/15・15/3・12/3）＋1〜180分の任意入力
- 作業終了音・休憩終了音を備え付け14音から個別に選択＋その場で試聴
- work→rest を自動で無限ループ
- 一時停止/再開・スキップ・リセット、進捗リング、macOS通知バナー
- 設定は自動保存、ログイン時の自動起動トグル、今日の完了ポモ数カウント

## ビルドと起動
```bash
./build.sh          # もしくは: bash build.sh
open build/Pomodoro.app
```
起動すると画面右上のメニューバーに 🍅 と残り時間が出ます。クリックで操作パネルが開きます。
終了はパネル右下の「終了」ボタンから。

## 必要環境
- macOS 14 以降（開発・確認は macOS 26 / Xcode 26 / Swift 6.3）
- Xcodeは開かず `swiftc` だけでビルドします

## 備考
- 音は `/System/Library/Sounds` にある備え付け音のみを使用します。
- 通知バナー・ログイン時起動は、初回に許可（通知）やログイン項目の承認が必要な場合があります。
- ローカルビルドのアドホック署名です（配布・審査は対象外）。
